# BraceletViewer
A tool for keeping track of the row you're on while making a friendship bracelet
<img width="1282" height="1032" alt="image" src="https://github.com/user-attachments/assets/39de4f6f-5c2c-40ee-9dae-10d2e0a2b815" />

# Features
- Open any bracelet file in image format (.png, jpg, etc.)
- Customizable overlays, don't like the pre-made ones? Just make your own and put them in the overlays folder!
- Customizable overlay start position. Does your bracelet design not fit with the default one I set? Just set your own start point!
- Customizable increment, if the overlay moves too much or not enough when you jump to the next line, you can just change it until it works!
- IT SAVES YOUR SPOT!!! If you close the program or your computer crashes or anything else happens, so long as you don't delete the save data file, your progress will be saved!

# Keybinds
- [Up Arrow] - Move up a row
- [Down Arrow] - Move down a row
- [Spacebar] - Reset to initial position

These all have on-screen buttons as well if you prefer that

# How to install
Well if you're a layman that doesn't have python installed on your computer, I've set up an .exe for you to make it very simple to use
1. Download the .zip file from [here](https://github.com/this-is-dev/BraceletViewer/releases/tag/Main)
2. Unzip it to the desired location
3. You're done! Open the .exe file and you're good to go :)

If you're a python user that doesn't want to use my .exe, you can just download the repository and run the .py file. The only dependency you should need to install manually is Pillow:
```pip install Pillow```

After that it should run like a dream
